# 📚 StudySync AI

An AI-powered student productivity web application designed to help students manage tasks, improve focus, and study smarter through intelligent academic assistance.

## 🌐 Live Demo

https://studysyncai-821eb.web.app

---

## 🚀 Overview

StudySync AI is a full-stack productivity platform built to solve common student challenges such as:

* Managing academic deadlines
* Organizing study tasks
* Staying focused
* Tracking productivity
* Receiving AI-powered study assistance

The platform provides each authenticated user with their own private workspace, ensuring task data and progress are securely isolated.

## 🔒 User Data Isolation

Each authenticated user has their own private task database.

Database structure:

users/{userId}/tasks

This ensures:

* No shared task visibility
* Private productivity tracking
* Secure user-specific data separation
